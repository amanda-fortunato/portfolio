Help Desk Documentation

Problema: Internet lenta

1. Verificar conexão física do cabo. 2. Reiniciar modem e roteador. 3. Executar comando "ipconfig /release" e "ipconfig /renew". 4. Testar conexão com "ping google.com".

Problem: Slow Internet

1. Check physical cable connection. 2. Restart modem and router. 3. Run "ipconfig /release" and "ipconfig /renew". 4. Test connection using "ping google.com".

Problema: Usuário não consegue fazer login

1. Confirmar usuário e senha. 2. Verificar se Caps Lock está ativado. 3. Resetar senha. 4. Verificar permissões da conta.

Problem: User cannot log in

1. Confirm username and password. 2. Check if Caps Lock is enabled. 3. Reset password. 4. Verify account permissions.
